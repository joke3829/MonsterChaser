
1. 실행 방법 ==================================================

게임 시작 전 먼저 아래 링크에서 실행에 필요한 리소스 파일들을 다운 받습니다.

클라이언트 리소스 파일
https://drive.google.com/file/d/1QCgWjl7ZBjCDZmhMN82MpkzA5qrswehr/view?usp=drive_link
(25.11.19 링크 확인 완료)

서버 리소스 파일
https://drive.google.com/file/d/1Mo672DJSLs0-th0OZBEskjqNM_KkDjuH/view?usp=drive_link
(25.11.19 링크 확인 완료)

다운 받은 두 파일을 압축 해제 후 

★ 클라이언트 리소스 파일을 압축 해제 후 나온 src 폴더를 
Client 폴더 안의 Client.exe와 같은 위치로 이동시킵니다.

☆ 서버도 마찬가지로 서버 리소스 파일을 압축 해제 후 나온 src 폴더를
Server 폴더 안의 Server.exe와 같은 위치로 이동시킵니다.


이후 서버로 사용할 컴퓨터에 Server.exe를 실행 시킵니다.
이제 Client.exe를 게임을 할 컴퓨터에 실행 후 서버 컴퓨터의 ip주소를 입력해 접속합니다.

2. 게임 진행 ==================================================

게임을 실행하면 타이틀이 나오고 화면을 클릭하거나 아무키를 누르면 
Select Room 화면으로 이동합니다.

Select Room에는 10개의 회색박스로 Room을 표시합니다. 원하는 Room을 클릭 해
게임 세션에 입장합니다.

입장 후 Select Job 버튼을 클릭 해 양옆의 화살표 클릭을 이용해 직접을 선택합니다.
원하는 직업이 나오면 Select를 클릭해 이전 화면으로 돌아갑니다.

R키를 눌러 Ready를 합니다. 모든 플레이어가 Ready하면 게임이 시작됩니다.

게임의 진행은 필드에 있는 보스 몬스터를 잡으면 다음 스테이지로 넘어갑니다.

필드에 있는 일반 몬스터들은 잡지 않아도 무방합니다.
(일반 몬스터는 사망 시 일정 시간 후 부활합니다.)

총 3개의 스테이지를 클리어 하시면 됩니다.

============================================================


3. 조작법 ======================================================

이동: W A S D + Shift(빠른 이동)
회피: Spacebar

카메라 회전: 마우스
줌 인, 줌 아웃: 마우스 휠

일반 공격: 좌클릭
스킬 사용: 왼쪽부터 Q E R

아이템 종류 변경: Z C
아이템 사용: X

Raytracing On/Off: B
Normal Mapping On/Off: N
Specular Check On/Off: M

1인칭 카메라: 9 
(1인칭 카메라 상태일 때 Spacebar와 Ctrl-L키로 카메라를 위 아래로 이동할 수 있습니다.)
3인칭 카메라: 0

스테이지 스킵하기: 7

전체화면 모드 On/Off: F9
프로그램 종료: ESC
